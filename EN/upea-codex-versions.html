⚠️ About security warnings when downloading or running

Your browser or operating system may display a warning indicating that the file could be malicious or that the publisher is unknown. Please be assured: this happens because the application does not have a paid digital signature. Originally, UPEA Codex was not planned as a downloadable application; its development arose to expand the UPEA Corporation project, offer greater convenience to collaborators, and take a further step in spreading our initiative.

It was never conceived as an ambitious project requiring a digital signature or massive hosting, but we decided to share it on Netlify so that more people could participate. Despite not having a paid signature, we publish the SHA256 hash of each file. You can verify the integrity and authenticity of the software by comparing the calculated hash with the one shown on this page. If they match, the file is legitimate and has not been altered.

We deeply appreciate your understanding and trust. We will continue to work with transparency and rigor.

How to verify the SHA256 hash

Windows (CMD or PowerShell)

certutil -hashfile "path_to_file.exe" SHA256 Get-FileHash -Algorithm SHA256 "path_to_file.exe"

Linux (terminal)

sha256sum path_to_file.AppImage

macOS (terminal)

shasum -a 256 path_to_file.dmg

Replace "path_to_file" with the path to the downloaded file. Compare the result with the SHA256 hash shown on this page. If they match, the file is authentic and has not been altered.
