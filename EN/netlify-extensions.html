Netlify Extensions at UPEA Corp.

Our website uses several Netlify extensions to improve security, performance, and functionality. Below, we detail the extensions relevant to the user experience:

User Agent Blocker

This extension allows us to block unwanted web trackers and malicious bots through specific rules in our robots.txt file or via Edge functions, which helps maintain the integrity and security of our content.

Bluesky Custom Domain

We use this extension to manage our custom domain on Bluesky, allowing us to maintain a consistent and recognizable presence on this platform.

Content Security Policy (CSP)

We implement a Content Security Policy that generates a unique script-src directive for each request. This is a crucial security measure that helps prevent script injection attacks and ensures that only resources from trusted sources are executed, thereby protecting our users from potential vulnerabilities.

Auth0 Integration [REMOVED]

UPEA Corp. integrates Auth0 to securely manage user authentication and authorization. This allows our users to register and log in efficiently and securely, ensuring that only authorized individuals can access specific functionalities, such as entity registration. We use this platform to provide login options with email/password, as well as social connections like Google or Facebook (if enabled), offering flexibility and security to our users.

These extensions are managed by Netlify, our web hosting provider, and contribute to UPEA Corp.'s technical infrastructure without requiring direct interaction from the end user.
