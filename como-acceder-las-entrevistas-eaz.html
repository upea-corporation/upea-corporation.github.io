Cómo Acceder a las Entrevistas

Para acceder a las entrevistas clasificadas de UPEA Corp., es necesario introducir la información correcta en los campos "Entidad" y "Código" del formulario de acceso. A continuación, se detalla la información requerida para cada campo:

  • Campo "Entidad": En este campo debes introducir el nombre de la entidad tal como está registrado oficialmente por UPEA Corp., y no su denominación común. Por ejemplo: EAZ#000
  • Campo "Código": Este campo funciona como una contraseña. Debes introducir el código de acceso confidencial asociado a las entrevistas de esa entidad específica. Este código garantiza que solo el personal autorizado o aquellos con la clave correcta puedan acceder a la información.

Una vez introducidos el ID de Entidad y el Código correctos, pulsa el botón "ACCEDER" para visualizar las entrevistas.
