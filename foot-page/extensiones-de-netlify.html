Extensiones de Netlify en UPEA Corp.

Nuestro sitio web utiliza varias extensiones de Netlify para mejorar la seguridad, el rendimiento y la funcionalidad. A continuación, detallamos las extensiones relevantes para la experiencia del usuario:

User Agent Blocker

Esta extensión nos permite bloquear rastreadores web no deseados y bots maliciosos mediante reglas específicas en nuestro archivo robots.txt o a través de funciones Edge, lo que ayuda a mantener la integridad y seguridad de nuestro contenido.

Bluesky Custom Domain

Utilizamos esta extensión para gestionar nuestro dominio personalizado en Bluesky, lo que nos permite mantener una presencia consistente y reconocible en esta plataforma.

Content Security Policy (CSP)

Implementamos una Política de Seguridad de Contenido que genera una directiva script-src única para cada solicitud. Esto es una medida de seguridad crucial que ayuda a prevenir ataques de inyección de scripts y garantiza que solo se ejecuten recursos de fuentes confiables, protegiendo así a nuestros usuarios de posibles vulnerabilidades.

Auth0 Integration [ELIMINADO]

UPEA Corp. integra Auth0 para gestionar de forma segura la autenticación y autorización de usuarios. Esto permite a nuestros usuarios registrarse e iniciar sesión de manera eficiente y segura, garantizando que solo las personas autorizadas puedan acceder a funcionalidades específicas, como el registro de entidades. Utilizamos esta plataforma para proporcionar opciones de inicio de sesión con correo electrónico/contraseña, así como conexiones sociales como Google o Facebook (si están habilitadas), ofreciendo flexibilidad y seguridad a nuestros usuarios.

Estas extensiones son gestionadas por Netlify, nuestro proveedor de alojamiento web, y contribuyen a la infraestructura técnica de UPEA Corp. sin requerir interacción directa por parte del usuario final.
